# AudioMind 用户手册

> 适用版本：V1.0

---

## 一、快速开始

### 1.1 访问系统

打开浏览器，输入地址（需问管理员获取）：
```
http://你的服务器IP:8080
```

### 1.2 界面介绍

```
┌──────────────────────────────────────────┐
│  🟢 AudioMind 课堂助手                     │  ← 标题栏
├──────────────────────────────────────────┤
│                                          │
│  AI: 你好！我是课堂录音助手...              │  ← 对话区
│  你: 老师布置了什么作业？                   │
│  AI: 根据课堂录音，老师布置了...            │
│                                          │
├──────────────────────────────────────────┤
│  📁 上传  [________________]  [发送]      │  ← 输入区
└──────────────────────────────────────────┘
```

---

## 二、上传课堂录音

### 步骤

1. 点击左下角 **📁 上传** 按钮
2. 选择课堂录音文件（支持 mp3/wav/m4a/webm/ogg）
3. 确认文件名和大小显示后，点击 **发送**
4. 等待转写完成（5 分钟录音约 30 秒）
5. 看到 ✅ 转写完成提示后，即可开始提问

### 注意事项

- 文件大小限制 25MB（约 30 分钟录音）
- 建议录音质量清晰、背景噪音小
- 一次上传一节课的录音效果最好

---

## 三、提问

### 直接输入问题

在输入框中输入问题，按 **Enter** 发送：
- "老师布置了什么作业？"
- "上节课讲了哪些重点？"
- "软件开发生命周期是什么？"
- "考试范围有哪些？"
- "帮我总结一下最近的课程内容"

### 查看回答

AI 会逐字显示回答，并标注信息来源。

---

## 四、管理知识库

点击顶部 **📋 知识库** 进入管理页面：

- **查看**：所有已上传的录音文档列表，含文件名、字数、上传时间
- **搜索**：输入关键词筛选录音
- **展开**：点击"展开全部 ↓"查看完整转写正文
- **删除**：点击 ✕ 删除不需要的录音文档

---

## 五、开始新对话

点击顶部 **＋ 新对话** 按钮：

- 清空当前聊天记录
- 重置对话上下文（AI 不再记忆之前的问答）
- 适合切换到不同课程的问题时使用

---

## 六、常见问题

**Q：上传后转写失败了？**
A：检查音频格式是否支持，文件是否损坏，网络是否正常。

**Q：问了一个问题但回答"未找到相关信息"？**
A：可能知识库中还没有相关录音。先上传包含该内容的课堂录音，再提问。

**Q：转写准确吗？**
A：使用 SiliconFlow SenseVoiceSmall 模型，中文准确率约 95%。清晰录音效果更好。

**Q：我的录音数据安全吗？**
A：转写文本存储在 Dify 云端知识库中，原始音频在转写后不保留。

**Q：怎样继续之前的对话？**
A：系统自动保存对话上下文（基于 conversation_id），重新打开页面后会继续之前的会话。想开始新话题点"＋ 新对话"即可。
